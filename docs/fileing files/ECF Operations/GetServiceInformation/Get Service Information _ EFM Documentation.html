Get Service Information

This sample demonstrates how request an electronic service list that contains the parties (or other persons current electronic service addresses) that have filed electronically in the case..

Sample Request   |   Sample Response

Request Parameter Description
CaseTrackingID Required. The CMS caseNumber. It's a publicly meaningful identifier usually displayed on public Court Calendars so people can answer the question, "what's your case number?". Retrieve this value from previous calls to GetCaseList or persisted in your own database.

NOTE: The use of searching by CaseTrackingID is incorrect in this implementation. In upcoming versions of this method, we will switch to the appropriate name CaseDocketId
